# Pre-Release Checklist

## Code Quality
- [ ] TypeScript compiles: `npm run typecheck`
- [ ] Tests pass: `npm test`
- [ ] Build succeeds: `npm run build`
- [ ] Validation passes: `npm run validate -- .`

## Functionality
- [ ] CLI commands work
- [ ] Detection engine works
- [ ] Installation modes work
- [ ] Backup/rollback works

## Documentation
- [ ] README.md updated
- [ ] CHANGELOG.md updated
- [ ] docs/ files accurate

## Package
- [ ] package.json version updated
- [ ] .npmignore configured
- [ ] LICENSE present

## Git
- [ ] All changes committed
- [ ] Git tag created

## Testing
- [ ] Unix/Linux tested
- [ ] Windows tested

## Security
- [ ] No secrets in code
- [ ] Attribution correct
