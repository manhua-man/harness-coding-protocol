# Planning / Review Bundle

## Purpose

This bundle supports structured planning, implementation review, and change inspection.

## Recommended Components

- Planning-oriented prompt or workflow
- Review checklist or critique pass
- Root-truth-aware agent instructions
- Optional issue or PR review helpers

## Good Fit For

- Scoped refactors
- Multi-file edits
- PR reviews
- Decisions that need traceability

## Boundary

- This bundle does not auto-install external planning tools
- It recommends a workflow combination; it does not replace the repository truth files
