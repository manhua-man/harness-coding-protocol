# Frontend Excellence Bundle

## Purpose

This bundle supports frontend-focused work where layout, interaction quality, and implementation detail matter.

## Recommended Components

- Frontend design guidance
- UI review or visual QA steps
- Component-level implementation workflow
- Root-truth-aware project instructions

## Good Fit For

- Product pages
- Application shells
- Dashboard and workflow UIs
- Iterative polish work

## Boundary

- This bundle recommends a strong frontend workflow, but it does not automatically install design systems or component libraries
- Visual checks should still be explicit and reviewable
