# MCP Productivity Bundle

## Purpose

This bundle groups tools that help an AI coding environment read, inspect, and act through MCP-style integrations.

## Recommended Components

- MCP context or documentation lookup
- Browser automation or verification tools
- GitHub or repository inspection helpers
- Root-truth-first instructions for safe use

## Good Fit For

- Tasks that need live context
- Cross-repo or cross-service lookup
- Tool-assisted verification
- Fast comparison of project state

## Boundary

- MCP tools are integrations, not the truth source
- The bundle recommends useful combinations, but it does not promise universal support for every MCP server
