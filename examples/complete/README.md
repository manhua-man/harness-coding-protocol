# Complete Example

This example shows what a fully installed repository can look like after enabling optional Cursor and Kiro adapters.

## Root Truth

- `AGENTS.md`
- `CLAUDE.md`
- `steering/`

## Compatibility Mirrors

- `.cursor/rules/harness-governance.mdc`
- `.kiro/steering/`
